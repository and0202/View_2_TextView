# View_2_TextView
学完这个个系列，自定义View就So Easy...
-------------------------------------------------------
第一课:
    继承View，实现基本TextView功能:更改内容和字体大小。
    
    使用到的知识点:
        1 TextPaint:存储字体大小、颜色等属性
        2 Canvas:画布，简单理解为可以在上面画一些东西
        3 View的绘制onDraw(Canvas canvas)方法
        4 TypedArray:一组容器用来保存属性的数组
        5 attrs:定义xml属性，例如: android:tex=""
